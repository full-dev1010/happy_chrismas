React Reconciliation

What is Reconciliation?

Reconciliation: The process which needs to figure out how to efficiently update the DOM when components "props" or "state" changes.

Example:

A component's render() function creates a tree of react elements.

When components "props" or "state" change, that render() function will return a different tree of react elements by updating UI DOM.

How reconciliation

Reconciliation process executes by diffing algorithm which has different set of rules.

What is diffing algorithm?

Component will update the DOM by an algorithm which compares DOM elements component's according to the behavior and in difference between them.

What are different set of rules in diffing algorithm?

  1. When DOM elements of different types
  2. When DOM element of the same type.
  3. Component element of the same type.
